JavaScript Prototypes Cheat Sheet

What is a JavaScript Prototype?

JavaScript prototypes are similar to Ruby Classes. Each JS Prototype has a Prototype Attribute which can be thought of as a characteristic of an Object. A JS Prototype Attribute will be inherited when a new instance of the prototype is created. If you create another separate prototype, it will have it's own prototype attributes, but you can choose to have it inherit Prototype Attributes from another prototype. In simple terms, a JS Prototype is like the blueprint or skeleton of a JS object. 

      function Animal(animalType, animalSound){
        this.type = animalType;
        this.sound = animalSound;
      }

      function Dog(dogName, dogBreed){
        this.name = dogName;
        this.breed = dogBreed;
      }

      Dog.prototype = new Animal("Dog", "Bark!");
      var Fido = new Dog("Fido","Terrier");

      Fido.sound // Should return "Bark!"

Using the Prototype Property

You can't add properties to a Prototype the same way you add properties to a JS Object. You can add properties to a Prototype by using the "Prototype Property".

          function person(first, last, age, eyecolor) {
              this.firstName = first;
              this.lastName = last;
              this.age = age;
              this.eyeColor = eyecolor;
          }
          person.prototype.nationality = "English";
          // This will add a nationality property to the person prototype.

Object.prototype.hasOwnProperty()

hasOwnProperty() detects if the called Object has that particular property and will return true. If the Object inherited the property it will return false.

          function Animal(animalType, animalSound){
            this.type = animalType;
            this.sound = animalSound;
          }

          function Dog(dogName, dogBreed){
            this.name = dogName;
            this.breed = dogBreed;
          }

          Dog.prototype = new Animal("Dog", "Bark!");
          var Fido = new Dog("Fido","Terrier");

          Fido.hasOwnProperty('breed'); // This should return true.
          Fido.hasOwnProperty('sound'); // This should return false.
